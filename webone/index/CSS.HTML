什么是CSS？

  1. CSS 指的是层叠样式表* (Cascading Style Sheets)
  2. CSS 描述了如何在屏幕、纸张或其他媒体上显示 HTML 元素
  3. CSS 节省了大量工作。它可以同时控制多张网页的布局
  4. 外部样式表存储在 CSS 文件中

CSS 是一种描述 HTML 文档样式的语言。

CSS实例 

                    
                    body {
                    background-color: lightblue;
                    }

                    h1 {
                    color: white;
                    text-align: center;
                    }

                    p {
                    font-family: verdana;
                    font-size: 20px;
                    }
